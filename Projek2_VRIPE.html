VR IPE

IPE VR is a VR application developed by FKKMK UGM in collaboration with Aruvana, where I worked as a software engineer. The application is designed to improve patient care training by providing a more immersive and interactive experience.

Tech Use

  1. Engine : Unity Engine 2021
  2. Plugin : XR Interaction Toolkit 2.0, BÃ©zier Path Creator, Photon PUN 2, and Obi Rope.

Story

This is my second project in Aruvana, I was tasked with developing an online VR quiz game with three different roles using Unity game engine and Photon PUN 2. The game must be playable by three people online, and the game will not start until all players are ready. I had to complete the game in 1 month and 2 weeks.
